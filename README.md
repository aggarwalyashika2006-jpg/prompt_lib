# 📚 Education Assistant Prompt Library for Teachers & Students

## 📌 Project Overview

This project is a **non-coding AI-based prompt library** designed to help teachers and students use ChatGPT effectively for educational tasks such as lesson planning, studying, assessments, and project work.

---

## 🎯 Objective

The main objective of this project is to create a **structured collection of prompts** that can simplify academic work and improve productivity using AI tools.

---

## 🛠️ Tools Used

* Microsoft Excel
* ChatGPT

---

## 📂 Project Files

* `prompt_library.xlsx` → Main dataset of prompts
* `work_report.pdf` → Detailed project report
* `demo_video.mp4` → Project demonstration 

---

## ⚙️ How It Works

1. The project stores **ready-to-use prompts** in an Excel sheet.
2. Users can select a prompt based on their need (teacher/student).
3. The prompt is copied and used in ChatGPT.
4. ChatGPT generates instant output such as lesson plans, quizzes, summaries, etc.
